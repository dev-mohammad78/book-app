# 📚 Book App

A modern and responsive **Book Application** built with **React**, **Vite**, and **Tailwind CSS**.

This project allows users to browse books, search through the collection, add or remove books from their favorites, and keep their favorite books even after refreshing the page using **Local Storage**.

This project was built as a learning exercise to improve React fundamentals such as component architecture, state management, props, conditional rendering, list rendering, and working with browser storage.

---

## ✨ Features

- 📖 Browse a collection of books
- 🔍 Search books by title
- ❤️ Add & remove books from favorites
- 💾 Persist favorite books using Local Storage
- 📱 Fully Responsive Design
- 🎨 Modern UI built with Tailwind CSS
- ⚡ Fast development with Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/book-app.git
```

Navigate into the project

```bash
cd book-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
